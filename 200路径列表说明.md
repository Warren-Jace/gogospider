## 💡 自定义路径

如需添加自定义路径，编辑 `core/common_paths.go`：

```go
var CommonPaths = []string{
    // 原有200个路径...
    
    // 🆕 添加您的自定义路径
    "/your-custom-path",
    "/special-endpoint",
    "/industry-specific-page",
}
```
